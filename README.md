# resume

#### 介绍
个人简历介绍：一个简洁美观的个人简历网页，包含个人信息模板和社交媒体链接。

#### 设计思路

- 使用卡片式布局，简洁现代
- 响应式设计，适配各种设备
- 包含个人信息模板和社交媒体链接
- 使用图标增强视觉效果

#### 功能特点
3.  **响应式设计**：适配各种屏幕尺寸
2.  **模块化布局**：个人信息、工作经历、技能、项目经验和社交媒体分块展示
3.  **社交媒体链接**：包含GitHub、CSDN、Bilibili和邮箱链接，带有平台图标
4.  **交互效果**：鼠标悬停时有轻微的动画效果
5.  **现代化设计**：使用卡片式布局和渐变背景

#### 使用说明

1. 将代码保存为HTML文件，直接在浏览器中打开即可
2. 修改个人信息部分，填入自己的实际内容
3. 更新社交媒体链接，替换为自己的账号链接
4. 如需添加个人头像，可以将头像部分替换为实际图片
